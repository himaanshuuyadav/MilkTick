# MilkTick

A modern Android application for dairy consumers to track milk deliveries, payments, and records.

## Features

- 📅 Daily milk delivery tracking
- 💰 Payment management and history
- 📊 Monthly and yearly analytics
- 📱 Custom notification system
- 🎨 Light/Dark theme support
- 📥 CSV export functionality
- 🚀 Onboarding experience

## Tech Stack

- Kotlin
- Jetpack Compose + Material3
- Firebase Firestore
- MVVM Architecture

## Setup

1. Clone the repository
2. Copy `.env.example` to `.env` and fill in your Firebase credentials
3. Add your `google-services.json` file to the `app/` directory
4. Build and run
